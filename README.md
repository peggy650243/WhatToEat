# 🎰 今天吃什麼？ (What To Eat Today?)

> **你的午餐救星！跟「選擇困難症」說再見。** > 不知道下一餐要吃什麼？把命運交給拉霸機決定吧！

## 📖 專案介紹 (Introduction)

你是否每天到了用餐時間，總是為了「要吃什麼？」而煩惱許久？
這款 **「今天吃什麼？」** Web App 專為解決你的選擇障礙而生。透過趣味的拉霸機介面，將枯燥的決定過程遊戲化。你只需要動動手指，我們就能隨機為你推薦今日美食，並直接帶你找到附近的餐廳！

## ✨ 主要功能 (Key Features)

* **🎯 自訂美食類別 (Smart Filtering)** 不想吃麵？只想吃飯？在開始之前，你可以先勾選或篩選出當下感興趣的食物類型（如：日式、泰式、速食、火鍋...等），確保隨機出來的結果都是你能接受的選擇。

* **🎲 趣味拉霸機制 (Slot Machine Randomizer)** 點擊按鈕，享受拉霸機轉動的期待感！系統會從你篩選的類別中，隨機選出一項最終結果。

* **📍 一鍵搜尋附近餐廳 (Find Nearby Restaurants)** 選定食物類型後（例如：拉麵），不必再手動切換 App。系統提供一鍵搜尋功能，直接幫你列出目前所在地附近的該類型餐廳，從「決定」到「出發」無縫接軌。

## 🚀 如何使用 (How to Use)

1.  **篩選類別**：在選單中勾選你今天「可能」想吃的幾種食物類型。
2.  **啟動拉霸**：按下「開始拉霸」按鈕，等待命運的安排。
3.  **搜尋餐廳**：滿意結果嗎？點擊搜尋按鈕，馬上查看附近哪裡吃得到！
